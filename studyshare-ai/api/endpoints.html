API Blueprint

StudyShare AI endpoint catalog

This document maps routes for authentication, notes, AI, rewards, admin moderation, and monitoring.

Auth endpoints

Method Path Purpose Protection
POST /api/auth/signup Create account with bcrypt password hash and email verify token Public + rate limit
POST /api/auth/login Validate credentials and create JWT session Public + failed login tracking
POST /api/auth/logout Revoke active session token User JWT
POST /api/auth/forgot-password Issue password reset token with expiry Public + throttle
POST /api/auth/reset-password Reset password via signed token Public token validate
POST /api/auth/verify-email Validate email verification token Public token validate

Notes and community endpoints

Method Path Purpose Protection
POST /api/notes/upload Upload file metadata and queue storage scan User JWT + MIME checks
GET /api/notes Search, filter, and paginate marketplace notes Public
GET /api/notes/:id Fetch note details, comments, and metrics Public / role aware
POST /api/notes/:id/like Like or unlike note User JWT
POST /api/notes/:id/save Save note to user shelf User JWT
POST /api/notes/:id/comments Add comment with sanitization and moderation pipeline User JWT + xss sanitize
POST /api/reports Create abuse report for note or user User JWT
POST /api/follow/:userId Follow or unfollow users User JWT

AI and rewards endpoints

Method Path Purpose Protection
POST /api/ai/chat Chat with context from selected notes User JWT + token quota
POST /api/ai/summarize Generate chapter summary User JWT + rate limit
POST /api/ai/mcq Generate MCQs with answer keys User JWT + quota
POST /api/ai/flashcards Create flashcard set from note sections User JWT + quota
POST /api/ai/quiz Create quiz set and difficulty levels User JWT + quota
GET /api/rewards/balance Read current token balance User JWT
POST /api/rewards/redeem Redeem tokens for premium feature unlock User JWT + wallet checks
GET /api/leaderboard Fetch global and scoped rankings Public

Admin and monitoring endpoints

Method Path Purpose Protection
GET /api/admin/users Paginated user moderation table Admin JWT
PATCH /api/admin/users/:id/status Ban or suspend user account Admin JWT + audit log
PATCH /api/admin/notes/:id/moderate Approve, reject, or remove upload Admin JWT + audit log
GET /api/admin/reports Review abuse reports queue Admin JWT
GET /api/admin/security-logs Read security events and suspicious activity Admin JWT
GET /api/monitor/metrics API latency, requests/min, error rate, db latency Admin JWT
