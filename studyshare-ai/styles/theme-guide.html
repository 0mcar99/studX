Design System

Theme and visual language guide

Blue, purple, and cyan gradient identity with glassmorphism surfaces and soft shadow hierarchy.

Color direction

Primary: #1d4ed8, Secondary: #6d28d9, Accent: #06b6d4, Surface whites and deep navy dark mode.

Typography

Display family Sora for headlines and Manrope for body copy to maintain SaaS-grade readability.

Motion cues

Page fade-in, staggered reveal, and shimmer skeleton states for perceived performance and polish.

Tailwind/shadcn implementation note

This prototype is intentionally authored with HTML + CSS only. In production Next.js implementation, map these utility patterns to Tailwind classes and shadcn primitives for component-level scalability.
