Architecture

Reference deployment design

Target architecture that maps directly to your preferred stack choices.

Frontend layer

Next.js App Router + React + TypeScript, styled with Tailwind/shadcn equivalents. Static prototype pages in this repository map route-by-route to production implementation.

Backend layer

Next.js API routes or Express services for auth, uploads, AI orchestration, rewards engine, moderation, and monitoring endpoints.

Data and storage

PostgreSQL (Neon/Supabase) with Prisma models, object storage on S3/Supabase/Cloudinary, plus signed URL access policies.

Authentication strategy

  • Email + password login with bcrypt hashes
  • Optional Google OAuth provider
  • JWT session tokens with expiry and revocation
  • Role-based route protection for Admin and User

AI layer

  • OpenAI API or local LLM adapters
  • Prompt templates by mode: summary, flashcards, quiz, MCQ
  • Context retrieval from selected uploads
  • Token metering and usage records in AIHistory
